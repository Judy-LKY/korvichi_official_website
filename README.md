# Korvichi Full Official Website

This is a static official website package for GitHub Pages.

## Upload instructions
1. Keep the `CNAME` file.
2. Upload `index.html`, `style.css`, `script.js`, `CNAME`, and `README.md` to the root of the GitHub repository.
3. GitHub Pages source should be `main` branch and `/(root)` folder.
4. Wait 1-3 minutes, then visit https://www.korvichi.com/

## Customize
Edit the contact information in `index.html`:
- Email
- WhatsApp
- Company address
- Product text

Add real product and factory photos later for a more professional result.
